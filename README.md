### Integer( %d or %i)
Integer defines a whole number
```
int a;
scanf("%d", &a);
```
### Unsigned Integer(%u)
Used to print unsigned decimal integers
```
unsigned int a;
scanf("%u", &a);
```

## Array
### Array Traversal

```
int v[n];
scanf("%d", &n);
for (int i=0; i<=n; i++)
{
    scanf("%d", &v[i]);
}
```
